# PWA Icons

Add these icon files here:

- `icon-192.png` — 192×192 px PNG
- `icon-512.png` — 512×512 px PNG
- `apple-touch-icon.png` — 180×180 px PNG
- `favicon.ico` — 32×32 ICO

You can generate them from a single source image using:
- https://realfavicongenerator.net
- https://maskable.app (for maskable icons)
